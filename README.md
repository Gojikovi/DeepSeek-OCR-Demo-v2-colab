# DeepSeek-OCR-2 Demo — Google Colab

Run DeepSeek-OCR-2 directly in Google Colab — no local setup required.

## 🚀 Quick Start

1. Open the notebook: [**DeepSeek_OCR_v2.ipynb**](https://colab.research.google.com/github/Gojikovi/DeepSeek-OCR-Demo-v2-colab/blob/main/DeepSeek_OCR_v2.ipynb)
2. Set runtime to **T4 GPU**: `Runtime → Change runtime type → T4 GPU`
3. Run all cells: `Runtime → Run all`
4. Wait for the public **Gradio Live link** to appear at the bottom — open it in any browser
